# DWBC Management System

A comprehensive management system built with **Laravel + Inertia + Vue.js** to handle employees, tasks, expenses, clients, and company data efficiently. It automates HR processes, task tracking, expense management, and client relationship management, providing admins with a complete dashboard for monitoring and reporting.

---

## Features
- Employee management with leave tracking and salary deductions  
- Task management with dashboards and overdue alerts  
- Expense management with approval workflow and reports  
- Company and commercial address management with profit tracking  
- Sales and client management with leads, quotations, and follow-ups  
- Admin dashboard with notifications and performance summaries  

---

## How to Use

Clone the repository:
 - Run `composer install`
 - Run `npm install`
 - Run `cp .env.example .env`
 - Run `php artisan key:generate`
- Set your database credentials in the .env file
 - Set your mail credentials in the .env file
 - Run `php artisan migrate`
 - Run `php artisan serve` to start the test server
